# 🍽️ Swiggy Restaurant Recommendation System

A personalized recommendation system built using Python and Streamlit that suggests restaurants based on user preferences like city, cuisine, rating, and cost.

## Features
- Data preprocessing with one-hot encoding
- Cosine similarity-based recommendations
- Interactive Streamlit web app

## File Structure
- `data/`: Contains raw, cleaned, and encoded datasets
- `models/`: Contains the saved encoder
- `app.py`: Streamlit application
- `preprocess.py`: Script to clean and encode the data
